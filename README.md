# AI_LAB

This repository contains AI laboratory assignments and projects covering various algorithms and machine learning techniques.

## Lab Structure

### Lab 1: Classification Problem
Implementation of classification algorithms and problem-solving approaches.

### Lab 2: Image Classification
Image classification tasks using computer vision and machine learning techniques.

### Lab 4: Heuristic Search Techniques
Optimization techniques including:
- **Greedy Algorithm Optimization**: Implementation of greedy algorithms for optimization problems
- **A* Search Technique**: Advanced heuristic search optimization using A* algorithm
